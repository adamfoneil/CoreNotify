CoreNotify

"Serverless" account confirmation and password reset emails for ASP.NET Identity

GitHub

Why CoreNotify?

  • ✅ Make ASP.NET Identity account emails work without setting up an email provider
  • ✅ No domain verification
  • ✅ No limit on number of apps you use with
  • ✅ Drop-in replacement for the default "NoOp" IEmailSender<T>.
  • ✅ CLI tool for easy setup and management
  • ✅ 30-day free trial, then $5/month

Getting Started

  1. Install the CoreNotify CLI:

    dotnet tool install --global CoreNotify.CLI

  2. Register your email:

    corenotify register you@example.com

    You should receive your API key.

  3. Add the NuGet package to your app:

    dotnet add package CoreNotify.MailerSend

  4. Configure your API key. See demo: appsettings.json

  5. Replace the default email sender

Things to Know

  • Uses MailerSend under the hood
  • Emails will show {yourdomain}.corenotify.net as the sender domain
  • Email content is fixed, see EmailSenderContent.cs
  • Learn more on the GitHub repo
